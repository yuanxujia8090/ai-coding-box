# Vibe Writing 参考资源

此文件夹包含 Vibe Writing 共创工作流的模板和参考文档。

---

## 📁 文件说明

### 聊天记录与素材.template.md

Vibe Writing 项目的核心模板，用于收集和组织素材、讨论记录、参考资源。

**包含内容：**
- 项目基本信息表
- 核心信息收集区（目标读者、分享场景、核心想法、参考样本）
- 素材收集区（每个素材包含：来源、原始内容、核心洞察、关键词）
- 讨论记录（时间、参与者、要点、决策、后续行动）
- 参考资源（文档链接、样本案例、竞品分析）
- 进度跟踪表
- 想法池

**使用方式：**
当项目初始化完成后，这个模板会被复制到你的项目目录中，作为 `聊天记录与素材.md` 的参考版本。

你可以：
1. 直接复制模板内容到实际工作文件
2. 参考模板结构组织自己的素材
3. 根据项目需求调整模板结构

## 🎯 工作流参考

完整的 Vibe Writing 工作流分为 5 个阶段：

| 阶段 | 主要任务 | 输出 |
|------|--------|------|
| Phase 0 | 项目初始化，确认关键信息，建立文件 | 项目目录 + 聊天记录与素材.md |
| Phase 1 | 素材收集，追问提炼，实时写入 | 素材文件内容 |
| Phase 2 | 结构规划，梳理逻辑，提出建议 | 文章/分享结构方案 |
| Phase 3 | 迭代写稿，根据反馈修改 | 初稿-v1、v2、vN.md |
| Phase 4 | 最终确认，选择输出形式 | HTML/Word/PDF |

详见主 SKILL.md 文档。

---

## 💡 最佳实践

1. **随时记录**：只要用户说了有价值的内容，就写入素材文件
2. **保留原声**：用户原话里的口语化表达和个人感悟是分享的灵魂
3. **追问提炼**：每个素材后面追问 1-2 个问题，挖掘更深洞察
4. **版本管理**：每个版本都存为独立文件，便于回退
5. **主动推进**：每个阶段结束后主动提示下一步
