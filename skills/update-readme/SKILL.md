此 skill 用于维护当前仓库的 README.md 索引，自动扫描并更新新增的文档链接。

## 使用场景

当用户在任意仓库目录下新增 .md 文件时，调用此 skill 自动更新该目录下的 README.md。

## 执行流程

1. **获取工作目录**: 检测当前工作目录（如 blog、ai-coding-box、frontend-interview-knowledge）
2. **扫描文件**: 使用 Glob 工具扫描当前目录下的 `**/*.md`（排除 README.md）
3. **提取标题**: 读取每个文件，提取首个 `#` 开头的标题作为显示名称
4. **读取现有索引**: 读取当前目录下的 README.md，解析已存在的链接
5. **智能合并**: 只添加不存在的链接，避免重复
6. **更新文件**: 使用 Write 工具更新 README.md

## 输出格式

**blog 仓库**（按日期分组）:
```markdown
# blog
记录日常问题、工具、阅读资料等

2026-03-25
- [文档标题](相对路径.md)

2026-03-24
- [文档标题](相对路径.md)
```

**ai-coding-box 仓库**（按目录分类）:
```markdown
# ai-coding-box
记录整理 AI 编程相关的资料、skill等

## skills
- [文档标题](相对路径.md)
```

**其他仓库**（简单列表）:
```markdown
# 仓库名称
仓库描述

- [文档标题](相对路径.md)
```

## 注意事项

- 自动检测仓库类型，采用对应的索引格式
- 新增文件默认使用当前日期分组（blog 仓库）
- 已存在的链接不会被重复添加
- 保持 README.md 的头部描述不变
